# Double Paste Preventer

A lightweight Windows tray app that prevents accidental double-pastes by blocking repeated Ctrl+V within a configurable time window.

## Features
- Configurable block duration (100ms – 10000ms)
- Enable/disable via tray icon or settings
- Run on startup option
- Minimal resource usage

## Requirements
- Windows Vista or later
- x86 or x64

## Usage
Run the executable. The app starts in the system tray. Left-click the tray icon to open settings, right-click for quick toggle and exit.
